# layerlane-apply

Personal job search assistant: a small **dashboard** (source of truth for experience, bullets, project stories, and analyzed roles) plus a **browser extension** that reads job postings and returns copy-ready outputs against that data.

## Status

Early scaffolding. Repo initialized; app code landing here next.

## Intended shape

1. **Dashboard / API** — CRUD for your evidence bank, job records, and JD analysis endpoints.
2. **Extension** — Scrape the active posting, send text to the backend, show fit summary and ranked bullets with copy buttons.
